👉 내가 처음에 시도한 방법
시작지점 (물이 새는 곳 -0.5) 의 배열과 끝나는 지점 (물이 새는 곳 + 0.5) 의 지점에 대한 배열을 각각 만들고
끝지점 - 시작지점 == 테이프의 길이 -> 테이프를 붙이고
끝지점 - 시작지점 > 테이프길이 ->  너무 멀리 떨어진 곳들 사이의 관계라고 생각해서 테이프를 각각 붙이고
끝지점 - 시작지점 < 테이프길이  -> 가까운 지점이므로 이미 테이프가 붙여졌다고 생각해서 continue
